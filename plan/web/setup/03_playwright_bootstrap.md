# Web Setup 03: Playwright Bootstrap

## Goal

Install and prove the Playwright browser-test foundation.

## Scope

- Database: not applicable.
- CLI: not applicable.
- Daemon: tests should target daemon-served HTTP behavior.
- YAML: not applicable.
- Prompts: not applicable.
- Tests: one browser smoke test must pass.
- Performance: browser startup should stay practical.
- Notes: document artifact paths and base Playwright config.

## Exit Criteria

- Playwright is installed.
- browser artifacts are configured.
- one browser smoke test passes.
