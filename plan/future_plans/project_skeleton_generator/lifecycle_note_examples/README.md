# Lifecycle Note Examples

These files are example starter lifecycle notes for a generated repository.

They are written as if they already live inside the generated repository's `notes/lifecycle/` folder.

The future generator should use them as:

- direct starter templates
- source material for templated rendering
- examples that the generated `AGENTS.md` and lifecycle overview explicitly reference

The intent is that an AI working in the generated repository can be told:

- read the lifecycle notes before making major changes
- follow the current stage note for required artifacts, verification posture, and completion boundaries
- use the example sections as the default shape unless the new repository deliberately revises them
