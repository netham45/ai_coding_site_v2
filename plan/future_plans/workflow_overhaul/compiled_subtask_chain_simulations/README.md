# Workflow Overhaul Compiled Subtask Chain Simulations

This directory contains planning-stage simulations of the compiled subtask chains expected for each draft workflow profile.

These are not implementation claims.

They are future-plan artifacts that bridge:

- starter workflow profiles
- prompt-contract expectations
- rigid step-order enforcement
- likely compiled workflow shape

## How To Read These Files

Each profile simulation shows:

- the profile id and node kind
- the expected compiled chain for that profile
- the main gate that blocks premature merge or completion
- the expected closure artifact or proof posture

These simulations are intentionally compact.

They do not replace:

- the starter profile YAML
- the prompt contract
- the broader flow specs under `flows/`
