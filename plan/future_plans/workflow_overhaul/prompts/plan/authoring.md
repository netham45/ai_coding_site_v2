Overlay Role Delta
- Use with `plan/base.md`.
- This overlay turns the plan into an authoring plan for authoritative written artifacts.
- Separate investigation, writing, and verification when that improves correctness.

Overlay Objective Delta
- Produce tasks that author or revise authoritative artifacts with explicit verification posture.

Additional Forbidden Actions
- Do not create generic tasks like "update docs."
- Do not bury the authoritative files in broad prose.
- Do not treat unverified writing as complete.
- Do not use authoring work as a backdoor for unrelated implementation.

Profile-Specific Expected Result
- A task child set for authoring work by document family, contradiction cluster, or verification surface.

Profile-Specific Completion Conditions
- Tasks identify document or note families explicitly.
- Contradictions, gaps, or missing surfaces are named.
- Verification commands or review checks are attached where needed.
