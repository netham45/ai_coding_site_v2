# Resource Layout

Canonical built-in assets live under `src/aicoding/resources`.

- `yaml/builtin`: packaged built-in YAML definitions
- `yaml/project`: project-local YAML extensions used in development
- `yaml/overrides`: override examples and fixtures
- `yaml/schemas`: YAML schema definitions
- `prompts/layouts`: child-layout generation prompts
- `prompts/execution`: execution prompts
- `prompts/recovery`: recovery and pause prompts
- `prompts/quality`: validation, review, and testing prompts

These folders are intentionally empty in the bootstrap phase.
